A feature-enhanced redesign of LinkedIn's Saved Posts page — built with React, TypeScript, and Tailwind CSS. Adds search with history, a stats dashboard, dark mode, per-post PDF export, and a revisit reminder — features the original LinkedIn doesn't have.
  # LinkedIn Saved Items UI

  This is a code bundle for LinkedIn Saved Items UI. The original project is available at https://www.figma.com/design/cnHBiIhfgqxdNpleoCzz8L/LinkedIn-Saved-Items-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
